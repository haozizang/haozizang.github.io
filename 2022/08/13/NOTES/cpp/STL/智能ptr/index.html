智能ptr

2022-08-13 10:01:59

  • 所有权

    • 通过 make_xxx 实现
    • 分类
      • 独占所有权
      • 共享所有权
      • 弱共享所有权

  • auto_ptr (deprecated)

    • copy会导致所有权被移走, 使用不便!

  • unique_ptr

    • 转移所有权, 用move
    • 存储在容器中
      • std::vector<unique_ptr<int>> v;
    • APIs
      • get() 获取关联对象的原生指针

  • shared_ptr

    • weak_ptr: 解决循环引用问题
      • weak_ptr.lock() -> 产生一个有访问权限的 shared_ptr
    • 不支持数组, 需要自己动手实现deleter(shared_ptr could not be used to manage dynamically allocated arrays, 但是cpp17以后有支持)
    • 实现
      • ptr1 -> 所管理数据的地址
      • ptr2 -> 执行控制块地址
        • 引用计数 use_count
        • 弱引用计数 weak_count
      • 计数的增加是线程安全, CAS技术
    • shared_ptr<Type>& 会带来线程不安全?

  • weak_ptr

    • 没有普通指针的行为 *, ->
    • 可从 shared_ptr / 另一个weak_ptr 构造

  • 智能指针使用守则

    • 指针混用(尽力避免)
    • 只管理堆上的对象
    • 优先使用 unique_ptr
      • 性能好/内存小
      • make_unique
    • 初始化 使用make_xxx
